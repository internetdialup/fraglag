# Fraglag 

A Python Regex based project for FPS-esque dictionary matching 

## Testing.

Run in terminal. Ensure latest python is installed. tmux python3 if needed. 

python3 demo.py // to display matching using the parser

## UTF-8, Enum, Dictionary.

Adjust the current sayings, remove, and add any in patterns.py ; Currently, acting as an enum at present. Trim if you want a tighter version, or create a dictionary.
